# 📸 Gallery Images

Members can upload event photos directly to this folder via GitHub.

## Folder Structure

| Folder | For |
|--------|-----|
| `cheti-chand/` | Cheti Chand celebration photos |
| `diwali/` | Diwali Gala photos |
| `picnic/` | Summer Picnic photos |
| `teejri/` | Teejri celebration photos |
| `community/` | General community events & gatherings |

## How to Upload

1. Go to the folder for your event (e.g., `cheti-chand/`)
2. Click **Add file → Upload files**
3. Drag & drop your photos
4. Click **Commit changes**

### Image Guidelines
- Use `.jpg` or `.png` format
- Keep file size under 5 MB per image
- Use descriptive names (e.g., `cheti-chand-2026-group-photo.jpg`)
